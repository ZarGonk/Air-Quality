# Air Quality
This code reads the city name entered by the user and displays the current weather conditions on the terminal

1. Visão Geral do Projeto
O projeto "Air-Quality", desenvolvido por ZarGonk e disponível no GitHub, é uma aplicação em Python projetada para fornecer informações sobre as condições climáticas atuais de uma cidade específica. O principal objetivo do projeto é demonstrar o uso de uma API para coletar e exibir dados meteorológicos diretamente no terminal do usuário.

2. Instruções de Uso da API (Baseado no Repositório)
Embora o repositório forneça o código-fonte, as instruções diretas de como a API é utilizada são inferidas do código, pois uma documentação explícita da API não estava disponível na página principal do repositório.

O projeto utiliza uma API externa para buscar os dados meteorológicos. Para utilizá-lo, siga os passos básicos:

Obtenção do Código:

Clone o repositório para o seu ambiente local:
Bash

git clone https://github.com/ZarGonk/Air-Quality.git
Navegue até o diretório do projeto:
Bash

cd Air-Quality
Configuração (Inferida):

É provável que o projeto precise de uma chave de API de um serviço de clima (como o OpenWeatherMap ou similar) para funcionar corretamente. Verifique o código-fonte (geralmente em um arquivo .py principal ou em um arquivo de configuração) para identificar onde a chave da API é necessária e como ela deve ser fornecida (ex: variável de ambiente, arquivo de configuração).
Instale as dependências Python necessárias (se houver). Você pode encontrar um arquivo requirements.txt no repositório. Se sim, instale-as com:
Bash

pip install -r requirements.txt
Execução:

O código lê o nome de uma cidade inserido pelo usuário. Para executar o script, use o interpretador Python:
Bash

python seu_script_principal.py
(Substitua seu_script_principal.py pelo nome do arquivo Python que contém a lógica principal do projeto, geralmente main.py ou um nome similar).
Após a execução, o programa solicitará que você insira o nome da cidade.
